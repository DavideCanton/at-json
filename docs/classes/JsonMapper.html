Static deserialize

  • deserialize<T>(ctor: Constructable<T>, source: any): T
    • Defined in mapper.ts:124

    Deserialization method. Deserialize jsonObj into an object built using ctor.

    It deserializes only properties decorated with some decorator from this library, or a custom one implemented using makeCustomDecorator .

    Annotated properties are deserialized into a property using the name value as the source name (defaults to the property name), if the mappingFn is present, it is invoked to allow deserialization customization.

    If complexType is specified, the property is treated as it had JsonComplexProperty decorator, recursively calling JsonMapper.serialize .

    If isArray is specified, the property is treated as it is an array, using respectively JsonArray or JsonArrayOfComplexProperty , according to other parameters.

Static serialize

  • serialize(source: JsonSerializable): any
    • Defined in mapper.ts:37

    Serialization method. Transform val into a new JSON value by applying the "serialization" step for each property decorator.

    It needs JsonSerializable implementation, and serializes only properties decorated with some decorator from this library, or a custom one implemented using makeCustomDecorator.

    Annotated properties are serialized into a property using the name value as the destination name (defaults to the property name), if the serializeFn is present, it is invoked to allow serialization customization.

    If complexType is specified, the property is treated as it had JsonComplexProperty decorator, recursively calling JsonMapper.serialize .

    If isArray is specified, the property is treated as it is an array, using respectively JsonArray or JsonArrayOfComplexProperty, according to other parameters.
